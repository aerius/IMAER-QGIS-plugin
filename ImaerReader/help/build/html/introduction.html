Introduction¶

About¶

This plug-in is developed by OpenGeoGroep in collaboration with the AERIUS team. The plug-in reads IMAER gml output from the AERIUS Calculator. How this is done is explained in the Usage section.

The plug-in is meant to be a demonstrator. So it does not have full functionality and is written to be easily extensible and adaptable (to eg. ArcGIS). How to extend or adapt this demonstrator is explained in the Developers section.

More information about AERIUS and IMAER (Information Model AERIUS) can be found on https://www.aerius.nl .
